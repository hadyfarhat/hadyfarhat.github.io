#3 - Battleship

This is a 2 player console game. Each player has ships placed in locations of his/her own choice. After placing the ships, the 2 players will play against each other by guessing the ship's location and hitting it.

Heavy knowledge of Object Oriented Programming is applied.

A terminal application that prepares better timesheets for a company.

The program writes and reads work data such as time spent on task, task completion, and other information in a CSV file.

This program contains CRUD operation functions that are found in most of today's websites.

#5 - Work Log DB

The Work Log project with added database.

This program uses an ORM to store and search data from a work log and prints detailed reports to the screen.

It has unit tests, a professional programming technique that helps ensure the quality of a program.
