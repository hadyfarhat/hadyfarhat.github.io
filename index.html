#1 - Movie Trailer

A program which stores a list of my favorite movies, including box art imagery and a movie trailer URL.

This data will then be served as a web page allowing visitors to review the movies and watch the trailers.

#2 - Soccer League

A program that helps a soccer coach divide 18 players into three well-balanced teams.

Python knowledge of CSV, lists, dictionaries is applied.

#3 - Battleship

This is a 2 player console game. Each player has ships placed in locations of his/her own choice. After placing the ships, the 2 players will play against each other by guessing the ship's location and hitting it.

Heavy knowledge of Object Oriented Programming is applied.

#4 - Work Log

A terminal application that prepares better timesheets for a company.

The program writes and reads work data such as time spent on task, task completion, and other information in a CSV file.

This program contains CRUD operation functions that are found in most of today's websites.

#5 - Work Log DB

The Work Log project with added database.

This program uses an ORM to store and search data from a work log and prints detailed reports to the screen.

It has unit tests, a professional programming technique that helps ensure the quality of a program.

#6 - Personal Learning Journal with Flask (A Python Microframework)

A web application which uses HTML, CSS, and Flask, a popular framework for Python web development.

This program, a blog-like web application, lets a user add and edit journal entries and store the results in a database.

#7 - Mineral Catalog with Django (Python Framework)

A website that displays information about various mineral (rocks) using the Django Framework.

The site will display a list of all the minerals in a database, with additional details available by clicking on specific minerals.
